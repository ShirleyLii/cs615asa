CS615A -- Aspects of System Administration - HW#4

Traceroute explained

Run the command traceroute www.stevens.edu on any Unix system on which you have sudo(8) privileges (for example, spin up an EC2 instance); observe the ICMP and UDP packets using tcpdump(8). Then, repeat the same process for traceroutes to to the following hosts to observe the different routes across the globe taken: www.uni-marburg.de, www.uba.ar, www.hawaii.edu, www.hku.hk, www.du.ac.in, www.usyd.edu.au.

Use Visual Trace Route and traceroute on a map to map the hops from different starting locations to these addresses. Report anything noteworthy or interesting. If anything appears surprising, research an explanation.
