CS615A -- Aspects of System Administration - Spring 2014 - Exercise: pkgsrc vs. pkgin

Exercise: Compare the use of pkgsrc and pkgin

Run a NetBSD EC2 instance

$ aws ec2 run-instances --instance-type t1.micro --image-id ami-65a7f30c

Binary Packages

$ /usr/bootstrap.sh binpkg
$ pkg_info | wc -l
$ man pkgin

Use pkgin(1) to install the nginx webserver.

When done, also run:

$ pkg_info | wc -l
$ pkg_info -D -a | more
$ /usr/pkg/sbin/pkg_admin fetch-pkg-vulnerabilities
$ /usr/pkg/sbin/pkg_admin audit

Source packages

$ /usr/bootstrap.sh pkgsrc
$ pkg_delete nginx pcre
$ cd /usr/pkgsrc/www/nginx
$ make install
$ pkg_info | wc -l

Build manually from source

$ ftp http://nginx.org/download/nginx-1.5.10.tar.gz
$ tar zxvf nginx-1.5.10.tar.gz
$ cd nginx-1.5.10
$ ./configure

Run a Fedora instance

$ aws ec2 run-instances --instance-type t1.micro --image-id ami-3b361952

Binary Packages

$ rpm -qa | wc -l
$ yum install nginx
$ rpm -qa | wc -l

Source packages

$ ftp ftp://ftp.netbsd.org/pub/pkgsrc/current/pkgsrc.tar.bz2
$ tar jxvf pkgsrc.tar.bz2
$ cd pkgsrc/bootstrap
$ sh bootstrap
$ cd ../www/nginx
$ bmake install

Build manually from source

$ curl http://nginx.org/download/nginx-1.5.10.tar.gz > nginx-1.5.10.tar.gz
$ tar zxvf nginx-1.5.10.tar.gz
$ cd nginx-1.5.10
$ ./configure
