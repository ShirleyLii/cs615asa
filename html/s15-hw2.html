CS615A -- Aspects of System Administration - HW#2

HW#2: Software packaging

Summary:

The objective of this assignment is for you to create a reliable, useful binary software package for a moderately complex piece of software with a few dependencies. The resulting package will be able to be installed on a newly created, RPM based EC2 instance simply by copying it to the host and running the appropriate rpm(1) command. The location of the files installed by the package will follow the file system hierarchy standard (ie. hier(7)), and best packaging practices.

Details

25 points

On a Fedora instance (ami-3b361952), create an RPM for the AWS command-line utilities, or "awscli" software. Verify that the resulting package can be installed on a new instance.

Deliverables and Due Date

As before, your deliverable is a document describing what you did to produce the RPM in a manner that would allow somebody else to reproduce the steps. In addition, you will provide the spec file from which a binary package can be created as well as the rpm(1) command(s) used to generate the package.

The files will be made available on your course folder under https://www.cs.stevens.edu/~your-username/CS615/HW2/. Please attach a copy of the files to an email sent from your @stevens.edu email address to jschauma@stevens.edu with a subject of "[CS615] HW2".
