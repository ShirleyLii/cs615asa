Traceroute explained

On any host you have access to, run the command traceroute www.cs.stevens.edu; observe the ICMP and UDP packets using tcpdump(1). Then, repeat the same process for traceroutes to to the following hosts to observe the different routes across the globe taken: www.uni-marburg.de, www.uba.ar, www.hawaii.edu, www.hku.hk, www.du.ac.in, www.usyd.edu.au.

Use Visual Trace Route and traceroute on a map to map the hops from different starting locations. Report anything noteworthy or interesting.

Amongst other things, You should observe that all traceroutes to Hawai`i appear to go through Colorado, rather than connect to the islands from the US West Coast directly Why is this?

Research the reasons for this and write a short summary of your findings. You probably want to revisit the topic of submarine network cables, peering agreements, backbone providers, etc.
